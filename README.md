# Synthetic Table Alignment

Dense tabular UI for alignment testing between headers and data rows.

## Scripts
- `npm install` to install dependencies
- `npm run dev` to start the dev server
- `npm run build` to create a production build
- `npm run preview` to preview the production build

## Tech
- Vite + React
- Tailwind CSS

## Notes
- Fonts load from Google (Space Grotesk).
- Tailwind scans `index.html` and `src/**/*.{js,jsx,ts,tsx}`.
